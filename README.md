# Odyssey Chronicle
This is odyssey Chronicle, a new desktop companion app for Elite Dangerous: Odyssey that is made with electron-forge, and thus cross-platform.
Mostly coded with Claude (as I had no electron/typescript experience) out of frustration with all the windows-based companion apps and how tedious they are to get working properly under Linux.

Biological species prediction data ported from EDMC-BioScan (https://github.com/Silarn/EDMC-BioScan/)
